API Reference
=============

Dartwork-mpl ships one import (`import dartwork_mpl as dm`) but several
small domains. Jump to the area you need—styles, layout, colors, fonts,
exports, or visual diagnostics—and each page explains the arguments in
plain language plus defaults that work in most cases.
